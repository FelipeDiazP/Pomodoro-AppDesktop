# 🕒 Pomodoro App Desktop — Versión 1.0.0

Una aplicación de escritorio minimalista desarrollada con **Electron.js** que implementa la técnica **Pomodoro**, ayudándote a mejorar tu concentración y productividad mientras estudias o trabajas.

---
⏱️ ¿Qué es la técnica Pomodoro?

La técnica Pomodoro divide el tiempo de trabajo en intervalos (por ejemplo, 25 minutos de enfoque y 5 minutos de descanso).
Después de cuatro ciclos, se recomienda un descanso más largo (15–30 minutos).

Este método ayuda a mejorar la concentración, evitar la fatiga mental y gestionar mejor el tiempo.
---

## 🚀 Características principales

✅ Temporizador Pomodoro funcional  
✅ Interfaz limpia y moderna  
✅ Notificación visual al finalizar cada sesión  
✅ Ventana fija (no se puede redimensionar ni mover)  
✅ Menú superior oculto para una experiencia enfocada  
✅ Instalador listo para Windows  

---

## 🧩 Tecnologías utilizadas

- **Electron.js** → para empaquetar la app de escritorio  
- **HTML5** → estructura visual  
- **CSS3** → estilos y diseño  
- **JavaScript (vanilla)** → lógica del temporizador y control de eventos
